⚙ Configuração necessária
Para usar o AgênciaOS, você precisa configurar suas credenciais do Supabase no arquivo HTML.
1. Abra o arquivo HTML em um editor de texto
2. Localize a linha: COLE_SUA_ANON_KEY_AQUI
3. Substitua pela sua chave anon do Supabase
4. Salve e reabra o arquivo
Sua anon key está em:
supabase.com → Settings → API → anon · public
